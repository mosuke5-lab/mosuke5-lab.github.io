  1. GET ‘/’ => ‘Hello’を返す
  2. GET ‘/search’ => GETパラメータ’q’のキーワードで検索した結果をjsonで返す
  3. POST ‘/update’ => 所定のjsonファイルをPOSTするとbulkでinsertされ、その結果をjsonで返す
